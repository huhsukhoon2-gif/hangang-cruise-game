# ⛴ 이랜드크루즈 한강 유람선 게임

여의도 **이랜드크루즈** 터미널을 출발해 **두물머리**까지 600 km를 항해하는 캐주얼 웹 게임입니다.  
실제 한강 사진을 배경으로, 바위와 유람선을 피하며 목적지를 향해 달리세요!

🎮 **[지금 바로 플레이하기](https://huhsukhoon2-gif.github.io/hangang-cruise-game/)**

---

## 🕹️ 조종 방법

| 키 | 동작 |
|---|---|
| `↑ ↓ ← →` / `W A S D` | 유람선 이동 |
| `ESC` | 일시정지 / 재개 |
| 터치 드래그 | 모바일 조종 |

## 🎯 게임 규칙

- **목숨**: ❤️ × 5
- **장애물**: 바위, 유람선, 대형 유람선
- **목표**: 여의도(0 km) → 두물머리(600 km) 완주
- 장애물과 충돌하면 목숨 1개 감소, **3초간 무적**
- 거리에 따라 속도 점점 증가

## ⭐ 난이도

| 난이도 | 장애물 속도 | 특징 |
|---|---|---|
| 쉬움 | 느림 | 초보자 추천, 여유 있는 플레이 |
| 중간 | 보통 | 표준 도전 |
| 어려움 | 빠름 | 대형 유람선 등장, 고속 장애물 |

## 🏞️ 랜드마크

항해 중 실제 한강 랜드마크를 통과합니다:
- 원효대교 → 마포대교 → 한강 중간 → 팔당대교 → 양수리 → 두물머리

## 🛠️ 기술 스택

- **HTML5 Canvas API** — 게임 렌더링
- **Vanilla JavaScript** — 외부 라이브러리 없음
